# NxsPPL-Finder
NxsPPL-Finder, or Nexus for short, is a small data-aggregator-collector project made in python.
## Function
Nexus is a data-aggregator, using python and various APIs to collect info in a profile format on given people(s).
## Installation
Clone the Repo.
